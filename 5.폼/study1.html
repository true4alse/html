form

사용자가 데이터를 입력할 수 있게 양식을 제공할 때 사용하는 태그

form태그는 action속성을 사용하여 사용자가 입력한 정보들을 보내주는 목적지를 설정할 수 있다

label태그를 통해 사용자가 입력하는 정보에 대한 이름표를 만들 수 있다.

사용자가 데이터를 입력할 수 있도록 입력박스를 만들 때 사용하는 태그는 input태그이며 type속성에 들어가는 값에 따라 여러가지 형태의 입력박스를 제공할 수 있다.

입력박스태그에 id라는 속성을 사용하여 고유한 이름을 지어준 뒤 연결할 label태그의 for속성에 동일한 이름을 저장하면 label태그와 입력박스태그는 서로 연결되어진다.

value라는 속성을 통해 입력박스에 제공되는 값을 넣어줄 수 있다.
