선택자의 우선순위 점수

선택자의 우선순위점수가 같다면 위에서 아래로의 해석흐름으로 인해 상대적으로 아래부분에 입력한 스타일이 최종 적용이 된다. 하지만 위에서 아래로의 해석흐름보다 선택자의 우선순위점수가 높은 스타일을 더 우선적용한다.

  • 태그명으로 선택 - 우선순위점수 1점
  • 클래스명으로 선택 - 우선순위점수 10점
  • 아이디명으로 선택 - 우선순위점수 100점
  • 인라인스타일방식 - 우선순위점수 1000점
  • css 에서 !important 키워드를 통해 스타일을 무조건 적용할 수 있다

선택자의 우선순위점수는 합산된다.
